#  FreeRTOS From Ground Up™ on ARM Processors (REVISED)

This repository contains a collection of **FreeRTOS-based embedded projects developed using STM32CubeIDE on FreeRTOS and STM32 Microcontrollers. These projects were implemented while completing the course FreeRTOS From Ground Up™ on ARM Processors.

The main objective of this repository is to demonstrate practical implementation of real-time embedded systems concepts on ARM Cortex-M microcontrollers. The projects cover various core features of FreeRTOS such as task management, scheduling, inter-task communication, synchronization, and interrupt handling.

Throughout these projects, different FreeRTOS APIs and mechanisms are implemented and tested using STM32 development boards and the STM32CubeIDE toolchain. The repository includes hands-on examples that help understand how multitasking firmware applications are designed and executed in real-time embedded environments.

Key concepts demonstrated in this repository include:

1. FreeRTOS task creation and scheduling
2. Task priorities and context switching
3. Inter-task communication using queues
4. Synchronization using semaphores and mutexes
5. Software timers and delays
6. Interrupt handling in FreeRTOS applications
7. Integration of FreeRTOS with STM32 HAL and peripherals

These projects are intended for learning, experimentation, and reference for embedded systems developers who want to understand how FreeRTOS works on ARM Cortex-M microcontrollers and how to build multitasking real-time applications. The course itself focuses on building real-time embedded applications, understanding RTOS internals, and mastering resource and task management features of FreeRTOS.
